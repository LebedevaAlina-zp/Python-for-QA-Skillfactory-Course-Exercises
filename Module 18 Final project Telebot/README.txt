# MyCurrencyCalcBot

Это телеграм-бот для конвертации валют.

Чтобы его запустить, нужно скачать проект, не забыть установить пакеты requests и pytelegrambotapi, 
и запустить модуль currencies_telebot_app.py.
Затем открыть в телеграме бот по имени @MyCurrencyCalcBot и следовать инструкциям, начиная с команды /start.

Выполнено в качестве итогового задания по разделу Python в рамках курса "Тестировщик-автоматизатор на Python (QAP)".
